send POST 

<?php		
$curl =curl_init();
$data = [
'number' => '628xxx',// number sender
'type' => 'chat',// type delivery
'message' => 'try message 1',// message content
'to' => ["628xxx","628xxx"], // number receiver
];

curl_setopt($curl, CURLOPT_CUSTOMREQUEST, "POST");
curl_setopt($curl, CURLOPT_RETURNTRANSFER, true);
curl_setopt($curl, CURLOPT_POSTFIELDS, json_encode($data) );
curl_setopt($curl, CURLOPT_URL, 'send');
curl_setopt($curl, CURLOPT_SSL_VERIFYHOST, 0);
curl_setopt($curl, CURLOPT_SSL_VERIFYPEER, 0);
curl_setopt($curl, CURLOPT_HTTPHEADER, ['Accept: application/json','Content-Type: application/json']);

$result = curl_exec($curl);
curl_close($curl);

print_r($result);
?>

Request Parameter

number Required sender phone number. You can use the country code prefix . Example: 628xxx
type Required type send
message Required Text message to be sent. Format: UTF-8 or UTF-16 string. for single newline (\n1 or PHP_EOL), double newline (\n2 or PHP_EOL PHP_EOL). maximum character is 1000
to Required receiver phone number. You can use the country code prefix . Example: 628xxx
